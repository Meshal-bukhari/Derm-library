Electrosurgery Quick Box

Electrosurgical Modalities — Rapid MCQ Review

One‑page, exam‑focused summary: choose modality by lesion depth & need for specimen.

Depth = Key
Quick Summary Box
Electrofulguration / ElectrodesiccationSuperficial epidermal ablation. Use for lesions confined to epidermis (e.g., SK, DPN, AK, lentigo, verruca plana). Low scarring risk when used properly.
ElectrocoagulationDeeper dermal destruction and robust hemostasis. Use for dermal lesions or arterial bleeding (e.g., large angiomas, BCC as part of C&E, SCC, sebaceous hyperplasia).
Electrosection (Cutting)Pure cutting current for incisions/excisions with minimal thermal artifact (best for histologic specimens, debulking, rhytidectomy incisions, shave removals when specimen required).
Other electrically‑mediated proceduresElectroepilation (hair removal: electrolysis, thermolysis), Iontophoresis (hyperhidrosis).
Indications — condensed (by modality)
Fulguration / Desiccation (superficial)
  • Skin tags (acrochordon), DPN, seborrheic keratosis
  • Actinic keratosis, lentigo, verruca plana
  • Small superficial angiomas
Coagulation (deep)
  • Large angioma/angiofibroma, telangiectasia
  • BCC/SCC as part of curettage & cautery
  • Ingrown toenail matrixectomy, sebaceous hyperplasia, syringoma
Electrosection (cutting)
  • When specimen needed (biopsy/excision) — minimal thermal artifact
  • Debulking (rhinophyma, AKN), cosmetic incisions (blepharoplasty), scar revision
Key procedural endpoints
  • Electrodesiccation endpoint: punctate bleeding = epidermis removed.
  • Fulguration = surface carbonization (sparking) for very superficial lesions.
  • Coagulation used for vessel control — expect scarring when dermis destroyed.
  • Electrosection (pure cutting) preserves histology; avoid blended currents if path needed.

Exam tricks & pitfalls What they test
  • Depth dilemma: Match modality to lesion depth. Superficial lesion → desiccation; dermal lesion → coagulation or excision.
  • Hemostasis trap: Questions describe bleeding. Capillary = electrodesiccation; arterial = electrocoagulation.
  • Specimen requirement: If histology needed, choose electrosection (pure cutting) or scalpel to avoid thermal artifact.
  • Over‑treatment: Using coagulation for an epidermal lesion leads to unnecessary scarring — common distractor.
Memory aids
  • Depth Ladder: Fulguration (surface) → Desiccation (epidermis) → Coagulation (dermis) → Section (cut through).
  • Mnemonic FEDS for superficial: Fulguration / Electrodesiccation — DPN, SK (superficial lesions).
Other procedures (one‑liners)
  • Electroepilation: Electrolysis/thermolysis for permanent hair removal (destroy hair root).
  • Iontophoresis: DC current for palmar/plantar axillary hyperhidrosis — likely ductal occlusion mechanism.
Quick recall checklist (for MCQs)
  • Is lesion superficial? → electrodesiccation/fulguration.
  • Do you need specimen? → electrosection or scalpel.
  • Type of bleeding? capillary vs arterial → desiccation vs coagulation.
  • Scarring acceptable? If no → prefer superficial or excision with minimal dermal damage.
