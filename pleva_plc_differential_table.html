🔬 PLEVA/PLC vs. Main Mimickers - Quick Reference

Condition Key Histologic Features Main Differentiating Points
PLEVA
(Acute)
  • "Wedge-shaped" lymphocytic infiltrate
  • Extensive necrotic keratinocytes
  • Dense interface dermatitis
  • Prominent RBC extravasation
  • Intraepidermal erythrocytes
  • Adnexal involvement
 BASELINE FEATURES:
  • Interface process + keratinocyte necrosis
  • NO true vasculitis (rare fibrinoid necrosis)
  • NO significant lymphoid atypia
PLC
(Chronic)
  • Milder lymphocytic infiltrate
  • Focal, scattered necrotic keratinocytes
  • Prominent parakeratosis
  • Mild interface changes
  • Mild RBC extravasation
 "PLC TRIAD":
  • 1️⃣ Interface infiltrate
  • 2️⃣ Focal necrotic keratinocytes
  • 3️⃣ Parakeratosis
Viral Exanthema
(e.g., Varicella)
  • Ballooning degeneration
  • Multinucleated giant cells
  • Interface dermatitis
  • Keratinocyte necrosis
 🔑 DIFFERENTIATORS:
  • ❌ NO wedge-shaped infiltrate
  • ❌ NO adnexal involvement
  • ✅ Ballooning + giant cells
  • ✅ Positive viral studies (PCR/DFA)
Guttate Psoriasis
  • Neutrophilic microabscesses (Munro's)
  • Regular acanthosis
  • Dilated papillary capillaries
  • Parakeratosis
  • Superficial perivascular infiltrate
 🔑 DIFFERENTIATORS:
  • ❌ NO interface component
  • ❌ NO necrotic keratinocytes
  • ✅ Neutrophils in stratum corneum
  • ✅ Regular epidermal thickening
Clinical: Small papules with silvery scale vs PLC's individual papules in crops
Small Plaque Parapsoriasis
(SPP)
  • "Nonspecific spongiotic dermatitis"
  • Epidermal edema (not cell death)
  • Focal parakeratosis
  • Mild perivascular infiltrate
 🔑 DIFFERENTIATORS:
  • ❌ NO true interface dermatitis
  • ❌ NO keratinocyte necrosis
  • ✅ Spongiosis > interface change
Clinical: Persistent stable patches >5cm or "digitate" shape vs PLC's recurrent individual papules
Hypopigmented MF
  • Cerebriform/atypical lymphocytes
  • Pautrier microabscesses
  • Prominent epidermotropism
  • Interface infiltrate
  • Pigment incontinence
 ⚠️ HIGH-YIELD TRAP:
  • ✅ Look for PLC triad first
  • ❌ NO Pautrier microabscesses in PLC
  • ❌ Epidermotropism out of proportion
  • ✅ Clonal T-cell population favors MF
Remember: Even hypopigmented variant of PLC shows the classic triad!

💡 Memory Tips:

  • PLEVA = "Destructive Wedge" - Dense, wedge-shaped infiltrate with extensive necrosis
  • PLC = "Smoldering Interface" - All features are "blunted" versions of PLEVA
  • Interface + Necrosis = Think Pityriasis Lichenoides - This combo rules out psoriasis and SPP
  • No True Vasculitis - RBC extravasation ≠ vessel wall necrosis
