Step Test Rationale / Key Points Source
Step 1:
Initial Diagnostic Evaluation		 Skin Biopsy for H&E Confirms diagnosis by showing vasculitis (leukocytoclasia & vessel wall necrosis) Page 430
Skin Biopsy for DIF Detects IgG, IgM, and C3 in vessel walls (70% cases); Granular BMZ pattern suggests SLE Page 431, Fig 24.19
CBC with Differential Assesses for anemia, leukocytosis, cytopenias Fig 24.19
Basic Metabolic Panel (BUN, Creatinine) Baseline renal function Fig 24.19
Urinalysis & LFTs Screens for renal and hepatic involvement (esp. infectious hepatitis) Fig 24.19
Step 2:
UV-Specific Stratification		 Serum Complement (C3, C4, CH50) Key prognostic feature. Low C4 suggests systemic involvement. Page 430; Fig 24.19
Anti-C1q Antibody Order if complement low. Suggests HUVS. Part of minor criteria for diagnosis. Page 430; Fig 24.19
Step 3:
Evaluate for Associated Diseases		 ANA Screen for SLE (positive in ~50% of HUVS) Page 431; Fig 24.19
Anti-ENA (anti-Ro, anti-La, anti-dsDNA, anti-Sm) Refine CTD diagnosis. HUVS rarely has anti-dsDNA/Sm – helps differentiate from SLE Page 431; Table 24.5
Rheumatoid Factor (RF) Screen for RA association Table 24.5
Hepatitis B/C Serologies Associated infections Table 24.5
HIV Serology Part of general vasculitis workup Table 24.5
SPEP/UPEP & IFE Screen for plasma cell dyscrasias (hematologic malignancies) Table 24.5
Peripheral Blood Smear Supports hematologic malignancy screen Table 24.5
Cryoglobulins Consider in hypocomplementemic cases Table 24.5; Fig 24.19
Chest X-ray & PFTs Indicated for patients with pulmonary symptoms Table 24.5
Echocardiogram & Cancer Screening As clinically indicated and age-appropriate. Look for occult malignancy. Table 24.5
