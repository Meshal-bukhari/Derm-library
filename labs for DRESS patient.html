DRESS Syndrome Workup & Longitudinal Monitoring

I. Initial Evaluation
Hematology
Tests
  • CBC with differential
  • Platelet count
  • Peripheral smear for atypical lymphocytes
To detect eosinophilia, atypical lymphocytosis, and monitor for cytopenias.
Renal Function
Tests
  • BUN, creatinine
  • Urinalysis
  • Spot urine for protein:creatinine ratio
Screen for interstitial nephritis (serious complication); rapid proteinuria detection.
Liver & Muscle Function
Tests
  • Liver function tests (LFTs)
  • Creatine kinase (CK)
  • Lipase
  • C-reactive protein (CRP)
To detect hepatitis, myositis, pancreatitis, and monitor inflammation.
Thyroid Function
Tests
  • TSH
  • Free T4
Establish baseline. Thyroid dysfunction is a key delayed complication.
Metabolic
Test
  • Fasting glucose
Baseline for corticosteroid use.
II. Additional Testing
Cardiac Evaluation
Tests
  • ECG
  • Troponin T
  • Baseline echocardiogram
Screen for myocarditis, a serious complication.
Virology (Pathogenesis)
Tests
  • Quantitative PCR for HHV-6, HHV-7, EBV, CMV
Investigate herpesvirus reactivation & predict late complications (e.g., CMV disease).
Exclusionary Tests (RegiSCAR Criteria)
Tests
  • ANA
  • Blood cultures
  • Hepatitis A/B/C serology
  • Chlamydia/Mycoplasma serology
Rule out autoimmune disease/sepsis. Help fulfill diagnostic scoring system (RegiSCAR).
Suspicion-Based Testing
If Interstitial Nephritis Suspected
  • Wright stain of urine for eosinophils (before corticosteroids)
If HLH Suspected
  • Ferritin
  • Triglycerides
  • LDH
  • Bone Marrow Examination
III. Longitudinal Monitoring & Referrals
Follow-up Strategy
  • Thyroid Function (TSH, Free T4): Repeat at 3 months, 1 year, and 2 years
  • HLH monitoring: Must be considered in follow-up (can evolve from DRESS)
  • Referral: Based on organ involvement; monitor for delayed complications over at least 1 year
